Pembaruan Sistem (OTA Web)

Unggah program ESP32 (.bin) secara nirkabel. Pastikan laptop/HP dan ESP32 berada di jaringan WiFi yang sama.

  1. Di Arduino IDE, klik menu Sketch ➔ Export compiled Binary.
  2. Cek IP Address ESP32 di Serial Monitor (atau setting router).
  3. Masukkan IP tersebut ke kolom di bawah, lalu klik Buka Portal.
  4. Pada tab baru, pilih file .bin tadi dan klik Upload.

Konfigurasi WiFi (Tanpa Kabel)

Jika ESP32 tidak terhubung internet, ikuti langkah berikut:

  1. Nyalakan daya ESP32.
  2. Buka pengaturan WiFi di HP/Laptop, cari jaringan bernama ESP32_BLDC_Config.
  3. Masukkan password: Admin123.
  4. Layar HP otomatis membuka Portal Setup. Klik tombol Configure WiFi.
  5. Pilih jaringan WiFi yang diinginkan, masukkan password WiFi tersebut, lalu klik Save.
  6. ESP32 akan merestart dan otomatis terhubung ke internet.
Prosedur Menjalankan Alat (SOP)
  1. Pastikan web terhubung ke broker MQTT. Buka menu Pengaturan Sistem, isi data berikut, lalu klik Connect:
    • Host: cc844c0fd3674e4cb628d1c05794ad09.s1.eu.hivemq.cloud
    • Port WSS: 8884
    • Username: bldcadmin
    • Password: Bldc_PNJ_123!
    Pastikan indikator MQTT di sudut kanan atas web berubah menjadi Online.
  2. Nyalakan saklar/MCB Power Supply 48V yang terhubung ke Motor Controller (ESC).
  3. Di menu Pengaturan, tentukan target putaran pada Setpoint RPM.
  4. Pilih jenis kendali (Mamdani atau Sugeno).
  5. Pilih kondisi pembebanan pengereman (Load 1 / 2 / 3 / 0).
  6. Tekan tombol ▶ Start Motor.
  7. Pantau kestabilan motor di tab Tipe Fuzzy atau Dashboard.
  8. Jika sudah selesai, wajib tekan tombol ■ Stop Motor.
